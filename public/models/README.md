# 3D Models

This folder contains 3D model files for objects in the escape room.

Recommended formats:
- .glb (recommended - compressed GLTF)
- .gltf (JSON-based 3D format)
- .fbx (for more complex models)

Example models needed:
- safe.glb
- painting.glb
- desk.glb
- key.glb

You can create or download models from:
- Sketchfab (https://sketchfab.com)
- Poly Haven (https://polyhaven.com)
- Blender (create your own)
