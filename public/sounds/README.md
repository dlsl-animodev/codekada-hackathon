# Sound Effects

This folder contains audio files for the escape room game.

Recommended formats:
- .mp3 (widely supported)
- .ogg (open format)
- .wav (uncompressed, larger files)

Example sounds needed:
- click.mp3 (object interaction)
- unlock.mp3 (puzzle solved)
- ambient.mp3 (background music)
- hint.mp3 (ai gives hint)
- error.mp3 (wrong answer)

Free sound resources:
- Freesound (https://freesound.org)
- Zapsplat (https://www.zapsplat.com)
- OpenGameArt (https://opengameart.org)
